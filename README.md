# PFE Master - Plateforme Adaptative d'Apprentissage des Langues

## Description
Plateforme intelligente pour l'apprentissage personnalisé des langues utilisant :
- Graph Neural Networks (GNN) pour la recommandation
- Large Language Models (LLMs) pour la génération de contenu
- Deep Learning pour l'adaptation au profil de l'apprenant

## Structure du projet

PFE MASTER/
├── notebooks/ # Notebooks d'expérimentation
├── src/ # Code source principal
├── data/ # Données (ajouter .gitkeep)
├── requirements.txt # Dépendances Python
